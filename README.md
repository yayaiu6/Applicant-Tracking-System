# Smart Hiring Hub – AI-Powered Applicant Tracking System (ATS)

**Live Demo:** [recru-ai.duckdns.org/admin](http://recru-ai.duckdns.org/admin)

Smart Hiring Hub is a complete recruitment platform that removes chaos from hiring. Companies post jobs, receive applications, and instantly see which candidates truly fit — ranked by AI with precise match scores from 1 to 100%. No more endless CV screening. No more guesswork.

## Why Companies Love It
- Hire faster with zero bias  
- Reduce screening time by 90%  
- Find hidden talent traditional ATS miss  
- Make every hiring decision data-driven

## How Applicants Use It
1. Visit the portal  
2. Filter by Department, Job Type, or Location  
3. Click “Apply”  
4. Upload CV (PDF ≤ 10MB) + details  
5. Get AI match score instantly

## How Employers Use It
1. Go to `/admin` → login  
2. Add/edit jobs  
3. Watch ranked applications pour in  
4. Export data or download CVs  
5. Hire the perfect fit in hours
